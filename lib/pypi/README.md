# voxlabs

Server-side voice synthesis and cloning library for Python.

## Installation

```bash
pip install voxlabs
```

## Usage

```python
from voxlabs import VoiceSynthesizer

synth = VoiceSynthesizer()
audio = synth.synthesize("Hello world")
```

## Features

- Server-side TTS
- Voice cloning
- Audio processing
- File storage
